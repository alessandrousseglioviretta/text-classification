Ghib.li Smart Text Classification

Text classification is the task is to assign a document to one or more classes or categories. When the categories are positive or negative, text classification is commonly referred to as sentiment analysis. Ghib.li offers a powerful and customizable machine-learning based engine to automatically classify multilingual text. I provide here two example applications.

Facebook comments

Facebook's Comment Box is a social plugin that allows users to add comments to web content. For example, have a look at Barack Obama's Plan on Facebook, which has received, at the time of writing, over 25'000 comments.

Clearly, reading thousands of comments one by one is not a viable solution to the problem of social media monitoring. I'm going to show how to retrieve and automatically classify Facebook comments using the Ghib.li classification engine. In this example, I'm going to use Python, a widespread general-purpose, high-level programming language. See the technical details on the wiki.

RSS feeds from Google Alerts

Google Alerts enables the creation of automatic email updates of the latest relevant Google results based on your queries. The updates are also offered as RSS feed. If the number of results is too large, you might want to filter the updates so to get only those that fit some additional criteria. With the Ghib.li classification engine, you can train a classifier to filter the news feed according to your wishes. See the technical details on the wiki.

Go to the Human Text Analysis Wiki and the Smart Text Classification Wiki for more details, or email info@ghib.li
