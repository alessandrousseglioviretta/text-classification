Facebook comments

Facebook's Comment Box is a social plugin that allows users to add comments to web content. For example, have a look at Barack Obama's Plan on Facebook, which has received, at the time of writing, over 25'000 comments.

Clearly, reading thousands of comments one by one is not a viable solution to the problem of social media monitoring. I'm going to show how to retrieve and automatically classify Facebook comments. In this example, I'm going to use Python, a widespread general-purpose, high-level programming language.

Get the comments

Facebook's comments are displayed in web browser in a human-friendly way, but they are also publicly available in a computer-friendly format called JSON. To retrieve them, we need to find out the Facebook-ID of the object (in our case, the image with President Obama's plan) to which the comments are attached. The URL of the image is the following: http://www.facebook.com/photo.php?fbid=10151435566046749. The Facebook ID is the long number that follows fbid=. If you now compose the following URL: https://graph.facebook.com/comments/?id=10151435566046749, you'll get a log list of comments in the following format:

 {
    "id": "10151435566046749_...",
    "from": {
        "name": "...",
        "id": "..."
        },
    "message": "...",
    "can_remove": false,
    "created_time": "2013-02-20T08:01:23+0000",
    "like_count": 0,
    "user_likes": false
}

Comments in this data format can be easily retrieved and parsed by using a few lines of Python:

import requests
import simplejson as json

FACEBOOK_GRAPH_COMMENTS_URL = 'https://graph.facebook.com/comments/'
params = {'id': '10151435566046749', 'limit': 10}

response = requests.get(FACEBOOK_GRAPH_COMMENTS_URL, params=params)
comments = response.json()

The comments object contains the latest 10 comments.
